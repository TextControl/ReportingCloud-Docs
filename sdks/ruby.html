ReportingCloud Ruby SDK

The ReportingCloud Ruby SDK makes it easy to interact with ReportingCloud from Ruby applications.

Installation

In order to use this wrapper, add this line to your application's Gemfile:

gem 'txtextcontrol-reportingcloud'

Or install it yourself as:

$ gem install txtextcontrol-reportingcloud
