ReportingCloud SDK Overview

Instead of consuming the plain Web API, we recommend the following language specific SDKs. They encapsulate all HTTP requests and provide classes to manage templates and to merge documents.

The following SDKs are supported:

  • .NET Framework
    The ReportingCloud .NET SDK NuGet package is a multi-target package and includes files for classic .NET, .NET Core and .NET Standard.

    .NET Framework SDK

  • ReportingCloud PHP SDK
    The ReportingCloud PHP SDK support all PHP versions better or equal to PHP 7.1.

    PHP SDK

  • ReportingCloud Java SDK
    The ReportingCloud Java SDK makes it easy to interact with ReportingCloud from Java applications.

    Java SDK

  • ReportingCloud Ruby SDK
    Building Ruby applications? The ReportingCloud Ruby SDK is the right choice then for your document generation processes in Ruby.

    Ruby SDK
