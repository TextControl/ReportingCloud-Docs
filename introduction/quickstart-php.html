ReportingCloud PHP SDK Quickstart Tutorial

The ReportingCloud PHP SDK makes it easy to interact with ReportingCloud from any PHP 7.1 or newer application. The most recent version of the SDK can be found on GitHub.

This quickstart tutorial shows how to create your first .NET application using ReportingCloud.

Prerequisites

  1. In order to use the ReportingCloud PHP SDK, you will need at least a trial account for ReportingCloud. How to register and to create a free trial account is described here: Sign up for your Free Trial Account

    Create Trial Account

  2. Create an API Key for your api calls. How API Keys are managed is described here: Manage API Keys

Your First Application

Since all versions of PHPs prior to 7.1 have reached end-of-life and consequently, no further security updates will be released for them, the PHP SDK only supports PHP 7.1 or newer. Check you are running PHP 7.1 or newer. Open your terminal and execute:

php --version

The output should be something like this:

PHP 7.1.26-2+0~20190216175340.13+stretch~1.gbpc20626 (cli) (built: Feb 16 2019 17:53:40) ( NTS ) Copyright (c) 1997-2018 The PHP Group Zend Engine v3.1.0, Copyright (c) 1998-2018 Zend Technologies with Zend OPcache v7.1.26-2+0~20190216175340.13+stretch~1.gbpc20626, Copyright (c) 1999-2018, by Zend Technologies with Xdebug v2.6.1, Copyright (c) 2002-2018, by Derick Rethans

Installation

Install the ReportingCloud PHP SDK in your project using Composer:

composer require textcontrol/txtextcontrol-reportingcloud:^2.0

You are now ready to create your first application.
